
# react-native-base-component

## 需要先把文件夹clone下来 

## 在package.json里加入 {file:../../react-native-base-component 为文件夹路径}
`"react-native-base-component": "file:../../react-native-base-component"`
## 然后执行npm install 或者 yarm install
